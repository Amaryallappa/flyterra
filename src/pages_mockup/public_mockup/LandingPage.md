# Landing Page Mockup

## Purpose
The primary entry point for all users. It introduces the AgriDrone service, its benefits, and the process of booking a drone spray.

## Functionality
- **Navigation**: Links to Features, How It Works, Pricing, Sign In, and Register.
- **Hero Section**: Catchy headline and "Book a Spray" call-to-action (CTA).
- **Features Grid**: Highlights Autonomous Spraying, Mapping, Scheduling, and Safety.
- **How It Works**: A 4-step guide (Register -> Verify -> Book -> Spray).
- **Pricing Card**: Shows a teaser price (e.g., ₹250/acre).
- **Footer**: Branding, Copyright, and legal links.

## Logic
- **Purely Presentation**: The page is static and contains no complex state or API calls.
- **Responsive Design**: Uses Tailwind grid and flexbox to adjust layout for mobile and desktop.

## API Interactivity
- None.
